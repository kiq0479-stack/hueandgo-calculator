# 휴앤고 단가계산기 - 요구사항 명세서

## 📋 프로젝트 개요
휴앤고 굿즈 제작 브랜드의 견적 자동화 시스템. 카페24 자사몰과 API 연동하여 상품/옵션/가격 정보를 실시간으로 가져오고, 견적서 및 거래명세서를 자동 생성한다.

## 🎯 핵심 문제
- 현재: 고객 문의 → 자사몰에서 가격 확인 → 수동으로 견적서 작성 → 캡처해서 전송
- 사이즈/수량 변경 요청 시 처음부터 다시 반복
- 시간 낭비 + 반복 작업
